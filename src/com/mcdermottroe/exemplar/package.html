Common utility code for the entire project.

This includes:

  • An {@link com.mcdermottroe.exemplar.Exception} base class for common exception handling.
  • Assertion/Design By Contract™ features.
  • Constants.
  • Java/XML string encoding methods.
  • Message formatting code.
  • Methods to help implementations of {@link java.lang.Object#equals(Object)}, {@link java.lang.Object#hashCode()} and {@link java.lang.Object#toString()}.
  • Package discovery code for dealing with plugin-style architectures.
